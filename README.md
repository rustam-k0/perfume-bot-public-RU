# Perfume Bot MVP

Бот для поиска парфюмов и их популярных клонов с расчетом экономии.  

## 📂 Структура 

```

perfume-bot/
│
├── data/
│   └── perfumes_master.csv   # единая база парфюмов 
│
├── bot.py                       # основной код Telegram бота
├── database.py                  # функции поиска и выдачи информации
├── utils.py                     # доп функции: логирование,
|                                   конвертация валют, fuzzy search
├── requirements.txt             # зависимости Python
├── .env                         # BOT_TOKEN и другие настройки
└── README.md                     # текущий файл

````

---

## 📝 Формат базы данных (CSV)

Единая база данных всех парфюмов и их клонов:

| Колонка                     Описание
--------------------------------------------------------------------------|
| `id`              | Уникальный идентификатор
| `brand`           | Бренд оригинального парфюма
| `name`            | Название оригинального парфюма
| `orig_price_eur`  | Цена оригинала в евро (примерно)
| `orig_market_url` | Ссылка на страницу оригинала
| `dupe`            | Наиболее популярный клон оригинала
| `dupe_price_eur`  | Цена клона в евро
| `dupe_market_url` | Ссылка на клон
| `saved_amount`    | Экономия в %: 
                      `(orig_price_eur - dupe_price_eur)/orig_price_eur*100

**Рекомендации по заполнению базы:**

- Сначала добавляйте бренды и названия парфюмов, потом цены и ссылки.
- Для MVP достаточно 50–200 записей; для реальной работы 500–1000 записей.
- Цены можно конвертировать из рублей или долларов в евро.
- Если нет ссылки — оставляйте `NA`.

## ⚡ Установка и запуск

1. Клонируйте репозиторий:
```bash
git clone <url>
cd perfume-bot
````

2. Установите зависимости:

```bash
pip install -r requirements.txt
```

3. Создайте файл `.env` с токеном бота:

```
BOT_TOKEN=ваш_токен_от_BotFather
```

4. Запуск бота:

```bash
python bot.py
```

* Бот будет онлайн пока работает скрипт.
* Для постоянной работы есть бесплатные хостинги: Replit, Railway, Render.

---

## 🔍 Как работает бот

1. Пользователь отправляет название парфюма.
2. Бот делает fuzzy search по базе данных.
3. Возвращает информацию о парфюме и одном популярном клоне:
   * Цена оригинала и клона
   * Ссылки на магазин
   * Процент экономии

**Пример ответа:**

```
Оригинал: Dior Sauvage
Цена: 120€
Ссылка: https://dior.com/sauvage

Клон: Armaf Derby Clubhouse
Цена: 30€
Ссылка: https://wildberries.ru/derby-clubhouse
Экономия: 75%
```

## 🛠 Подготовка базы данных

1. Соберите все исходные файлы (Excel, CSV).
2. Задайте одинаковые колонки:

```
brand
name
orig_price_eur
orig_market_url
dupe
dupe_price_eur
dupe_market_url
```

3. Объедините все файлы через Python (pandas) в один CSV (`perfumes_master.csv`).

4. Нормализуйте данные:

   * `brand` → title case, без лишних пробелов
   * `name` → strip, lower/upper по необходимости
   * Удалите дубли (`brand+name`)

5. Добавьте цены и ссылки, затем пересчитайте `saved_amount`:

```python
df['saved_amount'] = ((df['orig_price_eur'] - df['dupe_price_eur']) / df['orig_price_eur'] * 100).round(2)
```

## 📌 Подсказки и рекомендации

* Data quality > code quality: лучше меньше записей, но с точными данными.
* Ship fast, iterate faster: MVP для теста рынка можно запустить за неделю.
* Собирайте метрики: кто ищет что, сколько кликов, какие парфюмы популярны.
* Фичи после MVP:
  * Расширение поиска по категориям
  * Рейтинг/отзывы
  * Возможность добавить wishlist
  * Интеграция с другими платформами (Discord, WhatsApp, Web)
